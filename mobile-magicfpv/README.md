# Witaj w aplikacji Expo 👋

To jest projekt [Expo](https://expo.dev) stworzony za pomocą [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Rozpoczęcie pracy

1. Zainstaluj zależności

   ```bash
   npm install
   ```

2. Uruchom aplikację

   ```bash
   npx expo start
   ```

W konsoli pojawią się opcje otwarcia aplikacji w:

- [wersji deweloperskiej](https://docs.expo.dev/develop/development-builds/introduction/)
- [emulatorze Android](https://docs.expo.dev/workflow/android-studio-emulator/)
- [symulatorze iOS](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go) — ograniczonej piaskownicy do testowania aplikacji Expo

Możesz zacząć rozwijać aplikację edytując pliki w katalogu **app**. Ten projekt używa [routingu opartego na plikach](https://docs.expo.dev/router/introduction).

## Nowy czysty projekt

Kiedy będziesz gotowy, uruchom:

```bash
npm run reset-project
```

To polecenie przeniesie kod startowy do katalogu **app-example** i utworzy pusty katalog **app**, w którym możesz zacząć programować.
